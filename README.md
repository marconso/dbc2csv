# Conversor de arquivos `.dbc` para `.csv`.

Esse programa baixa o arquivo direto do diretório dentro do ftpdatasus e extrai para csv

<span style="color:red;"> Obs: Os arquivos .dbc e .dbf baixados durante o processo, serão excluidos após criação do .csv</span>

## Método de uso:

./dbc2csv /path_to_save_output /path_download_file.dbc
